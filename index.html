Research & Projects

Innovative solutions at the intersection of optics and computation

Lensless Microendoscopy System

Developed a minimally invasive brain imaging system that fits through an 18-gauge needle, using computational reconstruction instead of traditional optics. Achieved sub-cellular resolution through innovative coded aperture multiplexing.

Optics Design MATLAB Python Hardware Integration

TRUST Algorithm

Created a novel Transformer-Driven U-Net for Sparse Target Recovery that combines traditional optimization with deep learning. Achieved 40% improvement in reconstruction quality while maintaining real-time performance.

PyTorch Vision Transformers Compressed Sensing GPU Computing

Hyperspectral Imaging Pipeline

Implemented end-to-end processing pipeline for multispectral neuronal activity monitoring. Optimized computational efficiency for processing 10x more data while maintaining real-time constraints.
